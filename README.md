# UINProjectBaseSet
プロジェクトを開始する時のサンプル。
汎用extension, function, baseviewなどを含む。
テストコード、サンプルテストコード、UIテストサンプル有り。

### ■ Spec

##### Xcode:
version 7.0.1
##### Swift:
version 2.0
##### PackageManager:
Alcatraz
```
$ curl -fsSL https://raw.github.com/supermarin/Alcatraz/master/Scripts/install.sh | sh
```

### ■ Pod
#### - cocoapods導入  
```
プロジェクトディレクトリにて下記を実行
$ gem install cocoapods
$ pod setup
```
